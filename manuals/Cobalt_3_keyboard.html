Cobalt 3

Keyboard guide

by Leonardo Leoni 


Why this guide?
-----------------------
Cobalt 3 is a pocket computer with a QWERTY type keyboard, with some particularities.
Below the keyboard layout with the complete keymap.


---------------------------------------------------------------------------------
|   1   |   2   |   3   |   4   |   5   |   6   |   7   |   8   |   9   |   0   |  
---------------------------------------------------------------------------------
|   Q   |   W   |   E   |   R   |   T   |   Y   |   U   |   I   |   O   |   P   |  
---------------------------------------------------------------------------------
|   A   |   S   |   D   |   F   |   G   |   H   |   J   |   K   |   L   | ENTER |  
---------------------------------------------------------------------------------
|  CAPS |   Z   |   X   |   C   |   V   |   B   |   N   |   M   |SYMBOL | SPACE |  
---------------------------------------------------------------------------------



The key SYMBOL is not enabled in Cobalt 3, if you press it, '#' will be displayed.

The key CAPS is used to type special characters. You need to type before the key CAPS, and then the other key, together.
Hold the keys for one second to make it working.
Below there's the map of the layout of special characters when you type the key CAPS.


---------------------------------------------------------------------------------
|   !   |   @   |   #   |   $   |   %   |   ^   |   &   |   (   |   )   |   *   |  
---------------------------------------------------------------------------------
|   <   |   >   |       |       |       |       |   _   |   ;   |   "   |  DEL  |  
---------------------------------------------------------------------------------
|   \   |   |   |       |       |       |       |   -   |   +   |   =   |       |  
---------------------------------------------------------------------------------
|       |   :   |   '   |   ?   |   /   |   *   |   ,   |   .   |       |       |  
---------------------------------------------------------------------------------


Note that characters typed with the keyboard are lowercase.
